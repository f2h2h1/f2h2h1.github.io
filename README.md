## 在本地运行

- 用 vscode 或其它能渲染 markdown 的软件浏览
- 运行一个静态的服务器，然后在浏览器里打开
    1. php
        ```
        php -S 127.0.0.1:9012
        ```
    1. python
        ```
        python -m http.server -b 127.0.0.1 9012
        ```
- 使用 gitbook
- ~~其实直接用记事本打开也是可以的，只是没有图片和样式~~

## gitbook

### 依赖
- node 14.17
- npm 7.24
- gitbook-cli 2.3.2

### 安装 gitbook

```
npm install gitbook-cli -g
```

### 运行 gitbook

构建和运行，默认会占用 4000 端口
```
gitbook serve
```

只构建
```
gitbook build
```
